# Admin Guide

## Accessing the Dashboard
- **URL**: `https://bkjgroup.com/admin`
- Login with the credentials provided by the IT Administrator.

## Managing Content

### Blog
1. Navigate to **Blog > Posts**.
2. Click **New Post**.
3. Fill in the Title, Content (using the Rich Text Editor), and assign a Category.
4. Upload a Thumbnail.
5. Set Status to `Published` and Save.

### Services
1. Navigate to **Services**.
2. Add a new service with an icon, short description, and full details.
3. Use the reorder feature to adjust how they display on the frontend.

### Settings
- Navigate to **System Settings** to update:
  - Contact numbers (including WhatsApp).
  - Corporate emails.
  - Social media links.
  - Bank Account details for all subsidiaries.
