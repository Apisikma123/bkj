# QA & Testing Checklist

## 1. Responsiveness
- [ ] Displays correctly on Mobile (320px - 480px).
- [ ] Displays correctly on Tablet (768px - 1024px).
- [ ] Displays correctly on Desktop & UltraWide (1024px+).
- [ ] No horizontal scrolling on any page.

## 2. Performance (Lighthouse)
- [ ] Performance > 98.
- [ ] Accessibility = 100.
- [ ] Best Practices = 100.
- [ ] SEO = 100.
- [ ] Images are lazy-loaded and optimized.

## 3. Functionality
- [ ] WhatsApp CTA works correctly on mobile and desktop.
- [ ] Form submissions (Contact) are working.
- [ ] Admin panel CRUD operations work without errors.
- [ ] Language switcher updates the UI text correctly.

## 4. Accessibility
- [ ] Keyboard navigation (Tab) flows logically.
- [ ] All images have `alt` tags.
- [ ] Color contrast meets WCAG AA standards.
- [ ] ARIA labels are present on interactive elements (e.g., mobile menu button).

## 5. SEO
- [ ] Meta tags populate correctly per page.
- [ ] `robots.txt` is configured.
- [ ] `sitemap.xml` is accessible.
- [ ] JSON-LD Structured Data validates in Google Rich Results Test.
